# Extrator_URL

- URLs e Parâmetros;
- Fatiamento e Imutabilidade de strings;
- Métodos da classe *str*;
- Orientações a objetos;
- Expressões Regulares ou RegEx para identificar padrões;
- Métodos Especiais do Python.
